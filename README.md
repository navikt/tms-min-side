# tms-min-side
Container som setter sammen mikrofrontends for Min Side.

## Kom i gang
1. Bygg tms-min-side ved å kjøre pnpm run build
2. Start appen lokalt ved å kjøre pnpm run dev og pnpm run mock
3. Appen nås på http://localhost:4321/minside
